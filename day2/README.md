# Day 2: Tool-Using Agent

## Objective
To extend a rule-based agent to use external tools.

## Approach
- The agent selects a tool based on user input
- Each tool performs a specific task
- The agent calls the tool and returns the result

## Tools Implemented
- Calculator
- Weather (mocked)
- Text Summarizer

## Learning Outcomes
- Tool abstraction
- Function-based execution
- Modular design
