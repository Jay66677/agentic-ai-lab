# Day 1: Rule-Based AI Agent

## Objective
To build a simple AI agent using rule-based logic.

## Approach
The agent follows a pipeline:
Input → Decision → Action

- Input is taken from the user
- Intent is identified using keyword matching
- Action is executed based on intent

## Features
- Greeting response
- Date retrieval
- Basic calculation

## Learning Outcomes
- Understanding of agent architecture
- Rule-based decision making
- Modular programming
